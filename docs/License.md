# License

Raw2D is licensed under the Apache License 2.0.

```text
Copyright 2026 Aditya Nandlal
```

## What This Means

Apache-2.0 is an open-source license. People can use, copy, modify, and redistribute Raw2D, including in commercial projects, as long as they follow the license terms.

The important attribution requirement is simple: redistributed copies must keep the license and copyright notices.

## Required Files

Raw2D distributions include:

```text
LICENSE
NOTICE
```

If someone redistributes Raw2D or a modified version, they should keep those files with the distribution.

## Attribution

The `NOTICE` file identifies the original Raw2D project:

```text
Raw2D
Copyright 2026 Aditya Nandlal
Original project: https://github.com/bestmaa/raw2d
Public documentation: https://raw2d.com/doc
```

This helps prevent forks or copies from removing the original author attribution.

## Project Name

The Apache-2.0 license covers the source code. It does not grant trademark rights.

The Raw2D name and project identity should not be used to imply that a fork, package, product, or service is the official Raw2D project unless permission is granted.

## Practical Rule

Allowed:

- use Raw2D in an app
- modify Raw2D
- fork Raw2D
- publish a fork with clear attribution

Not allowed:

- remove the original copyright notice
- remove the `LICENSE` or `NOTICE` files from redistributed copies
- present a fork as the official Raw2D project
- claim the original Raw2D work was authored by someone else
