# Raw2D v1.23.4 - MCP Studio Automation Phase

Raw2D v1.23.4 publishes the MCP Studio automation phase.

## Added

- MCP scene edit plans for Studio-safe create, update, delete, reorder, and asset-reference operations.
- Studio scene JSON validation for assets, commands, duplicate IDs, and renderer-specific warnings.
- Generated Studio examples that round-trip through save/load and renderer adapter checks.
- English and Hinglish documentation for MCP Studio automation boundaries and agent-safe workflows.
- MCP README coverage for edit plans, Studio validation, generated examples, and tool boundaries.

## Notes

- MCP tooling stays JSON-first and reviewable. It does not click Studio UI, write files by default, push Git, or publish packages.
- Studio keeps ownership of UI state, command history, persistence, and renderer adapter execution.
- Canvas remains the reference renderer while WebGL warnings stay explicit for texture and renderer caveats.
- Package versions are aligned at `1.23.4` across every published Raw2D package.
- npm publish and Cloudflare deployment should be performed by the GitHub tag workflow or a maintainer with credentials.
