# Raw2D v1.14.6 - Studio Persistence Phase

Raw2D v1.14.6 publishes the Studio persistence phase.

## Added

- Stable Studio scene JSON save with deterministic `.raw2d.json` filenames.
- Studio scene load with schema validation and user-visible import errors.
- Canvas PNG export from the Studio viewport.
- Studio persistence documentation in English and Hinglish.
- Browser smoke coverage for Studio save, load, and export wiring.

## Notes

- Studio persistence stays inside `apps/studio`; runtime packages still do not import editor code.
- The scene JSON format currently stores object data, camera state, and renderer mode.
- PNG export reads the active Studio canvas and does not add asset packaging yet.

## Verification

- `npm test`
- `npm run build:docs`
- `npm run test:browser`
- `npm run pack:check -- --silent`
- `npm run test:consumer`
- Browser checks for Studio save, load, export, and invalid import error handling.
