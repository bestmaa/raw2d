# Raw2D v1.11.8 - Studio Shell Phase

Raw2D v1.11.8 publishes the first Raw2D Studio shell phase.

## Added

- `apps/studio`, a strict Vite TypeScript editor app scaffold.
- Studio workspace layout with topbar actions, tools, canvas workspace, layers, properties, and renderer controls.
- Canvas/WebGL renderer switch UI for the editor shell.
- Studio scene JSON state with camera and object list.
- Runtime adapter that converts Studio scene state into Raw2D scene, camera, and drawable objects.
- Sample scene action that renders Rect, Circle, and Text2D through the Raw2D Canvas renderer.
- Studio smoke tests for scene state, render adapter, and visual shell wiring.

## Notes

- Studio is still an MVP shell. Drawing tools, selection, persistence, and export are planned in later TASK4 phases.
- The Studio app stays separate from the library packages so Raw2D core remains lightweight.

## Verification

- `npm --prefix apps/studio run typecheck`
- `npm --prefix apps/studio run build`
- `npm run typecheck`
- `node --test tests/studio/*.test.mjs`
- Browser check for `/studio/` with Sample scene and renderer switch.
