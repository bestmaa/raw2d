# Raw2D v1.13.8 - Studio Interaction Phase

Raw2D v1.13.8 publishes the Studio interaction phase.

## Added

- Single-object selection in `apps/studio`.
- Drag movement for selected objects.
- Rect and Sprite resize handles.
- Keyboard move, delete/backspace, and escape-clear commands.
- Layers panel with select, visibility, and ordering actions.
- Properties panel for transform, geometry, text, and material edits.
- Renderer Stats panel backed by Canvas stats and WebGL diagnostics.
- Studio interaction and panel documentation in English and Hinglish.

## Notes

- Studio remains an app under `apps/studio`; runtime packages still do not import editor code.
- Sprite still uses an `assetSlot` placeholder until persistence and asset import land.
- Renderer stats are read-only diagnostics and panels do not draw objects directly.

## Verification

- `npm test`
- `npm run build:docs`
- `npm run test:browser`
- `npm run pack:check -- --silent`
- `npm run test:consumer`
- Browser checks for Studio selection, drag, resize, keyboard, layers, properties, and stats.
