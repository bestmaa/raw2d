# Raw2D v1.7.10 - Public Beta Hardening

Raw2D v1.7.10 completes the Public Beta Hardening phase. The focus is not new rendering features; it is proving that a fresh user can install, read, copy examples, and verify the docs before the next public beta push.

## Highlights

- Fresh install audits for the umbrella package, focused Canvas/WebGL packages, and `raw2d-react`.
- CDN smoke page and pinned jsDelivr verification for post-publish checks.
- Product docs snippet audit that copies TypeScript snippets into a temporary app and typechecks them.
- Browser bug-bash, mobile viewport, and dark UI overflow checks for docs and examples.

## Added

- `/cdn-smoke` route for pinned CDN checks.
- `npm run test:cdn:pinned`
- `npm run test:snippets:product`
- `npm run test:browser:mobile-docs`
- `npm run test:browser:dark-overflow`
- English and Hinglish docs for beta install audits, CDN smoke, product snippets, browser bug bash, mobile viewport, and dark overflow.

## Verification

- `npm test`
- `npm run build:docs`
- `npm run pack:check -- --silent`
- `npm run test:consumer`
- `npm run test:browser`
- `npm run test:browser:mobile-docs`
- `npm run test:browser:dark-overflow`
- `git diff --check`
