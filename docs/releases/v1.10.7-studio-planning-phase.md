# Raw2D v1.10.7 - Studio Planning Phase

Raw2D v1.10.7 publishes the Studio planning phase for the future visual editor.

## Added

- Studio scope and non-goals docs.
- Studio app/package boundary docs.
- Studio scene JSON save/load design.
- Studio tools design for select, move, resize, shape, text, and sprite.
- Studio panels design for layers, properties, assets, and renderer stats.
- `/studio` placeholder route without runtime renderer coupling.
- `STUDIO_TASKS.md` for the Studio MVP implementation queue.

- CDN smoke docs point to the `1.10.7` pinned package URLs.
