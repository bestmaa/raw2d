# Raw2D v1.24.5 - Studio Advanced Editing Phase

Raw2D v1.24.5 publishes the Studio advanced editing phase.

## Added

- Studio grouping and ungrouping with visible `Group2D`-style hierarchy and child world-position preservation.
- Studio duplication, alignment, distribution, and snap-to-grid helpers as explicit commands.
- Studio zoom-to-selection, fit-to-scene, and read-only minimap helpers for larger scenes.
- Stable `raw2d-studio-clipboard` copy and paste payloads with validation, id remapping, and safe asset metadata.
- English and Hinglish documentation for advanced editing workflows and keyboard accessibility.

## Notes

- Advanced Studio editing stays in `apps/studio`; runtime packages still receive scene state through the adapter.
- Grouping, arrangement, paste, and navigation helpers keep Canvas and WebGL rendering concerns separate from editor commands.
- Clipboard payloads are document JSON, not canvas pixels or a project packaging format.
- Package versions are aligned at `1.24.5` across every published Raw2D package.
- npm publish and Cloudflare deployment should be performed by the GitHub tag workflow or a maintainer with credentials.
