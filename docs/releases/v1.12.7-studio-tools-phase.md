# Raw2D v1.12.7 - Studio Tools Phase

Raw2D v1.12.7 publishes the first Studio drawing tools phase.

## Added

- Studio Rect, Circle, Line, Text2D, and Sprite placeholder creation tools.
- Scene JSON factories for each drawing tool.
- Dynamic Studio Layers and Properties output for created objects.
- Sprite placeholder objects with an `assetSlot` field for future texture or atlas wiring.
- Studio docs with small and full examples for tool behavior.
- Unit tests for tool scene JSON output and runtime adapter coverage.

## Notes

- Sprite still renders as a placeholder Rect in Studio until asset loading is added.
- Tool clicks mutate Studio scene state and then redraw through the Raw2D runtime adapter.
