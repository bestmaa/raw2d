# Raw2D v1.8.9 - Beginner Docs Phase

Raw2D v1.8.9 completes the beginner docs phase. The focus is clearer first-run learning, stronger docs search, polished Hinglish copy, and beginner-friendly MCP guidance.

## Highlights

- Start Here flow now has coverage checks for install, Canvas, Scene, Camera, shapes, texture/sprite, WebGL, interaction, MCP, and React.
- Docs search handles common spelling mistakes and short user queries.
- Hinglish docs use cleaner sentences instead of rough word-by-word translation.
- MCP beginner docs explain why `raw2d-mcp` is separate from the runtime package.
- Next/previous docs navigation now shows group progress and stable topic metadata.

## Verification

- `npm run test:browser`
- `npm run pack:check -- --silent`
- `npm run test:consumer`
