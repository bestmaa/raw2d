# Raw2D v1.10.8 - Release Pipeline Fix

Raw2D v1.10.8 fixes the release pipeline after v1.10.7 failed before npm publish.

## Changed

- Package versions are aligned at `1.10.8`.
- Vite is pinned to the stable `7.3.x` line for predictable installs.
- CDN smoke docs point to the `1.10.8` pinned package URLs.

## Fixed

- GitHub Actions publish no longer depends on Vite 8/Rolldown native binding install behavior.
- Cloudflare builds no longer hit the stale lockfile entry for unavailable `@oxc-project/types@0.17.0`.

## Verification

- `npm ci --ignore-scripts`
- `npm run build:docs`
