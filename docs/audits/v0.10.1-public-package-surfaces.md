# Raw2D v0.10.1 Public Package Surface Audit

Date: 2026-05-03
Task: T090

## Scope

This audit checks that Raw2D packages expose stable entry points without leaking
private implementation helpers through package barrels.

## Package Entry Rules

- Every publishable package exposes only the root export path `"."`.
- Focused packages use `./dist/index.js` and `./dist/index.d.ts`.
- The umbrella `raw2d` package exposes ESM and UMD bundles.
- The umbrella package does not expose renderer internals, MCP tools, or React bridge APIs.
- Focused packages may expose low-level helpers only when they are part of that package identity.

## Private Helper Denylist

- Interaction internals: `InteractionControllerActions`, `InteractionScope`, `pointerCapture`
- MCP generation internals: `generateRaw2DExampleHelpers`
- React bridge internals: context, prop appliers, and scene-object hooks
- Sprite packing internals: packer stats and item sorting helpers
- Umbrella internals: Canvas object renderer, WebGL buffers/caches/batchers, MCP, and React bridge
