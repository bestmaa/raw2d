# Raw2D v0.10.3 Package Metadata Audit

Date: 2026-05-03
Task: T092

## Scope

This audit checks package README, license, notice, trademark, and npm metadata
for every publishable Raw2D package.

## Rules

- Every package keeps `README.md`, `LICENSE`, `NOTICE`, and `TRADEMARKS.md`.
- Every package publishes those legal files in the npm tarball.
- Every package uses `Apache-2.0` and author `Aditya Nandlal`.
- Every package links to the Raw2D GitHub repository, issues page, and docs.
- Every package has discoverable npm keywords including `raw2d`.

## Audited Packages

- `raw2d`
- `raw2d-core`
- `raw2d-canvas`
- `raw2d-webgl`
- `raw2d-sprite`
- `raw2d-text`

- `raw2d-react`

## Verification

```bash
node --test tests/package/metadata-audit.test.mjs

```

The package metadata audit now fails if a package drops attribution, legal
files, npm discovery fields, repository links, or trademark notice packaging.
