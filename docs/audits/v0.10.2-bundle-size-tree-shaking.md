# Raw2D v0.10.2 Bundle Size And Tree-Shaking Audit

Date: 2026-05-03
Task: T091

## Scope

This audit records the current bundle footprint and adds a repeatable guard for
tree-shaking-friendly package metadata.

## Current Build Output

- `packages/raw2d/dist/raw2d.js`: about 145.5 kB, about 33.9 kB gzip.
- `packages/raw2d/dist/raw2d.umd.cjs`: about 111.0 kB, about 29.7 kB gzip.
- Focused package entry files stay thin and under 12 kB each.

## Tree-Shaking Contract

- Publishable packages keep `sideEffects: false`.
- Focused package entry files remain small barrel files.
- The umbrella package is convenient, while focused packages remain the better
  choice for users who want smaller app bundles.

## Repeatable Check

```bash
npm run build
npm run audit:bundle
```

The audit fails when the umbrella bundles exceed current guardrail thresholds,
when focused entries grow too large, or when a package loses `sideEffects: false`.
