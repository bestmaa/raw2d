# Raw2D Marks

Raw2D is the name of the original project maintained by Aditya Nandlal.

The Apache-2.0 license grants permissions for the source code, but it does not grant trademark rights. Do not use the Raw2D name, logo, or project identity in a way that suggests your fork, package, product, or service is the official Raw2D project unless you have permission.

Allowed uses:

- referring to Raw2D as a dependency
- saying that a project is built with Raw2D
- linking to the original Raw2D project
- making clear that a fork is a fork

Not allowed:

- presenting a fork as the official Raw2D project
- removing attribution and claiming the original Raw2D work as your own
- using the Raw2D name or branding to imply endorsement
